Welcome to CSV QuickView!

Just one more step to enable lightning-fast CSV viewing.

Action Required: Grant File Access

For your security, Chrome requires you to manually grant extensions permission to access local files.

To complete setup:

  1. Click the button below to open the Extensions page.
  2. Find CSV QuickView in the list.
  3. Turn on the toggle switch for "Allow access to file URLs".

After enabling this setting, you can simply double-click any .csv file on your computer to open it instantly in Chrome.

This extension will only use this permission to read and display .csv files you open in the browser.
